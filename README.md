💱 Currency Converter Web App

A modern, responsive, and feature-rich Currency Converter built using HTML, CSS, and JavaScript.
It fetches real-time exchange rates and allows users to convert between currencies instantly.

🚀 Features

🌍 Real-time currency conversion

🔄 Swap currencies with animation

⚡ Fast API-based exchange rates

📱 Fully responsive design

🎨 Modern UI with animations (Dark + Gold theme)

❗ Input validation with error handling

📊 Exchange rate details + last updated time

🔢 Smooth number count-up animation

🛠️ Tech Stack

HTML5 – Structure

CSS3 – Styling & Animations

JavaScript (Vanilla JS) – Logic & API handling

ExchangeRate API – Real-time currency data
